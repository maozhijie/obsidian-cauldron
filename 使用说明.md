# 丹道修炼系统 · 使用说明（v0.2.0）

> **将每日生产力行为化作修仙之旅。** 完成任务采集"药材"，番茄钟积累"药引"，每日自动"封炉炼丹"产出专属"丹药"，再将丹药投注于项目、模板与角色，积累修炼经验突破境界——你的每一天，都是一炉修炼。

---

## 核心玩法循环

```
完成任务 → 采集药材 + 药引
         ↓
使用番茄钟 → 积累专注引
         ↓
每日 23:00 → 封炉炼丹（含陈化/配伍/丹纹判定）
         ↓
产出丹药 → 投注到项目/模板/角色
         ↓
积累修炼XP → 境界提升 → 解锁新功能
```

## 安装方法

### 手动安装

1. 下载以下三个文件：`main.js`、`manifest.json`、`styles.css`
2. 在你的 Vault 目录下找到 `.obsidian/plugins/` 文件夹
3. 创建名为 `dandao-cultivation` 的子文件夹
4. 将三个文件复制到该文件夹中：
   ```
   <你的Vault>/.obsidian/plugins/dandao-cultivation/
   ├── main.js
   ├── manifest.json
   └── styles.css
   ```

### 启用插件

1. 打开 Obsidian，进入 **设置 → 社区插件**
2. 确保已关闭"安全模式"（受限模式）
3. 在已安装插件列表中找到 **丹道修炼系统**，点击开关启用

## 核心概念

### 领域标签 & 性味

**领域标签**是你为任务划分的成长维度（比如"学术写作"、"运动"、"编程"等），每个标签对应一种**性味**——就像修仙中药材的属性：

| 性味 | 图标 | 含义 | 示例领域 |
|:---:|:---:|------|--------|
| 神识 | 🧠 | 思维、认知类活动 | 学术写作、阅读 |
| 根骨 | 💪 | 体魄、基础类活动 | 运动、健康 |
| 灵动 | ⚡ | 技巧、创造类活动 | 编程、绘画 |
| 融合 | 🔮 | 沟通、协作类活动 | 社交、团队协作 |

插件默认预设了四个领域标签，你可以在设置中自由增减。

### 药材

**完成任务 = 采集药材。** 当你在笔记中勾选一条带有领域标签的任务时，系统会自动采集一份对应领域的药材。

- 普通任务 → 采集 **良材** × 1
- 高优先级任务（带 🔺 或 ⏫ 标记）→ 采集 **珍材** × 2

药材稀有度共四级：凡材 → 良材 → 珍材 → 异材。

### 药引

**药引**是炼丹的辅助材料，影响丹药的品级和纯度。共有四种：

| 药引 | 图标 | 获取方式 |
|:---:|:---:|------|
| 守时引 | ⏰ | 在截止日期前完成任务 |
| 延时引 | ⏳ | 超过截止日期后才完成任务 |
| 专注引 | 🧘 | 完成一个番茄钟工作时段 |
| 断念引 | 💔 | 提前中断番茄钟 |

> 小贴士：守时引和专注引越多，丹药品质越好；延时引和断念引会降低品级哦。

### 封炉炼丹

每日到达封炉时间（默认 23:00），系统会自动将当天采集的药材和药引投入丹炉，经过"配伍"计算，炼出一枚丹药。

炼丹规则：
- **君药**：当日数量最多的药材领域，决定丹药的主性味和名称
- **臣药**：数量次多的领域（需达到君药数量的 30% 以上），为丹药增添辅性味
- 如果当日没有采集任何药材，只会标记封炉，不会产出丹药

### 丹药

每日封炉的产出，拥有以下属性：

**品级**（由药引组合决定）：

| 品级 | 说明 |
|:---:|------|
| 凡品 | 基础品质 |
| 灵品 | 有一定专注和守时 |
| 宝品 | 高度专注且守时 |
| 神品 | 极致专注、完美守时 |

**纯度**（由守时引与延时引的比例决定）：
- 纯度 = 守时引 ÷ (守时引 + 延时引)
- 范围为 0.3 ~ 1.0，按时完成的任务越多，纯度越高

**命名规则**：
- 有臣药：`【主性味·辅性味·品级】君药领域丹`，如"【神识·灵动·灵品】学术写作丹"
- 无臣药：`【主性味·品级】君药领域丹`，如"【灵动·宝品】编程丹"

## 投注系统

炼出的丹药并非摆设——你可以将丹药**投注**到三个方向，获取不同维度的增益反馈，让修炼成果真正作用于你的生产力体系。

### 项目投注（短期增益）

将丹药投注于当前活跃的项目，可为该项目带来**药材稀有度 +1 级**的临时加成：

| 丹药品级 | 加成持续天数 |
|:---:|:---:|
| 凡品 | 1 天 |
| 灵品 | 3 天 |
| 宝品 | 5 天 |
| 神品 | 7 天 |

你可以在**投注 Tab** 中创建和管理项目。

### 模板投注（中期复利）

将丹药投注于重复任务模板，可令该模板的**守时引效果倍增**：

- 每次投注为该模板增加 **+0.1 倍率**
- 投注等级随累计投注次数成长，复利效应显著

### 角色投注（长期成长）

将丹药投注于自定义角色（如"创作者"、"学习者"、"修行者"等），角色将获得经验值：

- XP 计算 = 品级权重 × 纯度
- XP 累计达标后角色自动提升等级
- 共 **10 级**等级体系，长期陪伴你的成长之路

## 播种层

灵感转瞬即逝，播种层助你以最低摩擦捕获每一颗灵感种子，并通过目标管理系统追踪长期进展。

### 快速播种

- 通过命令面板 `Ctrl/Cmd+P` → **快速播种**，弹出极简输入框
- 一键记录灵感种子，零摩擦捕获
- **种子 Tab** 内也提供内联输入框，随手播种

### 种子池

- **种子 Tab** 展示所有待处理的灵感种子
- 每颗种子支持两种操作：
  - **转化为任务**：指定目标文件路径，将种子正式纳入任务体系
  - **丢弃**：清理不再需要的种子
- 超过设定天数（默认 7 天）未处理的种子会显示**过期提醒**，避免灵感遗忘

### 目标管理

- 在**种子 Tab** 底部可管理目标
- 设定目标名称、目标值、单位（如"读完 12 本书"）
- 手动更新进度，进度条可视化呈现达成情况

## 外丹体系深化

v0.2.0 大幅丰富了外丹炼制体系，新增药材陈化、配伍相冲/冲和、丹炉养成、丹纹系统与多周期丹炉。

### 药材陈化

药材采集后并非永久保鲜，而是拥有**有效期**（默认 7 天，可在设置中调整），经历四个阶段：

| 阶段 | 标识 | 颜色 | 说明 |
|:---:|:---:|:---:|------|
| 新鲜 | fresh | 🟢 绿 | 刚采集的药材 |
| 陈化中 | aging | 🟡 黄 | 正在陈化，品质稳定 |
| 成熟 | mature | 🟣 紫 | 陈化完成，部分增值领域药材稀有度提升 |
| 过期 | expired | ⚪ 灰 | 已过有效期，不参与炼丹 |

> 小贴士：某些增值领域的药材在陈化至"成熟"后，稀有度会自动提升一级，善用陈化可获得更好的炼丹材料。

### 配伍相冲与冲和

不同性味的药材同炉时存在**相冲**关系：

- **神识** 🧠 与 **根骨** 💪 相冲
- **灵动** ⚡ 与 **融合** 🔮 相冲

相冲效果：
- 相冲药材同炉 → 丹药品级**降低**
- 但当冲突药材比例在 **40%~60%** 之间时，触发**"冲和"** → 品级反而**提升**

> 正所谓"对立统一、阴阳调和"，冲和之道是高阶炼丹师的必修课。

### 丹炉养成

丹炉不再是静态工具，而是跟随你一同成长的伙伴：

- 丹炉通过每次炼丹积累经验，共 **10 级**
- 高等级丹炉提供：
  - 品级加成
  - 纯度下限保底
  - 丹纹概率提升
- 在**炼丹 Tab** 中可查看丹炉等级和经验条

### 丹纹系统

高品级丹药有概率出现**丹纹**——额外的天赋印记，带来独特增益：

| 丹药品级 | 丹纹概率 |
|:---:|:---:|
| 宝品 | 10%+ |
| 神品 | 30%+ |

丹纹分三档：

| 档次 | 示例 | 额外效果示例 |
|:---:|------|------------|
| common | 灵光纹、清心纹 | 投注效果 +10% |
| rare | 混元纹、归元纹 | 投注效果 +20%、修炼经验 +15% |
| legendary | 太极纹、天道纹 | 修炼经验 +30%、全属性加成 |

### 多周期丹炉

除了每日自动封炉的**日炉**外，你还可以创建更长周期的丹炉：

| 炉型 | 周期 | 说明 |
|:---:|:---:|------|
| 周炉 | 7 天 | 以周为单位的长线炼丹 |
| 月炉 | 30 天 | 以月为单位的深度炼制 |
| 项目炉 | 跟随项目 | 与项目生命周期同步 |

- 长周期丹炉自动分配 **20%** 的日常药材
- 通过命令面板 `Ctrl/Cmd+P` → **管理周期炉** 打开管理界面

## 内丹体系

外丹炼制之上，更有**内丹修炼**——通过持续炼丹积累修炼经验，突破境界，解锁更强大的功能。

### 修炼境界

五大境界，每境九层，层层递进：

| 境界 | 层数 | 解锁内容 |
|:---:|:---:|------|
| 练气 | 1~9 | 基础功能：药材采集、日炉炼丹 |
| 筑基 | 1~9 | 解锁周炉、投注系统 |
| 金丹 | 1~9 | 解锁自定义丹纹、月炉 |
| 元婴 | 1~9 | 更高阶功能 |
| 化神 | 1~9 | 终极境界 |

- XP 来源：每日封炉炼丹，按丹药品级获得经验
- 在**修炼 Tab** 中查看当前境界与进度

### 经脉系统

每个领域标签映射为一条**经脉**，持续在某领域投入可提升该经脉等级：

- 共 **10 级**：未通 → 初通 → 小成 → … → 合道
- 每级提供 **5% 采集加成**，满级即 50% 加成
- 在**修炼 Tab** 中以网格形式展示所有经脉状态

### 境界突破

当境界层数达到 **9 层**时，可尝试突破至下一大境界：

**突破条件**（需全部满足）：
- 连续修炼天数达标
- 累计炼丹数达标
- 持有特定品级丹药
- 经脉均衡度达标

**成功率计算**：
- 基础成功率 = 70%
- 最终成功率 = 70% + 心境值 × 5%（最高 99%）
- 失败后积累心境值，**保证最终必定成功**

> 突破时会触发仪式感弹窗动画，以"渡劫"之姿迈入新境界。通过命令面板 `Ctrl/Cmd+P` → **尝试突破** 开启突破仪式。

## 任务格式

### 基本格式

在任何 Markdown 笔记中，使用标准任务语法并添加领域标签即可：

```markdown
- [x] 完成论文第三章初稿 #学术写作
- [x] 跑步 5 公里 #运动
- [ ] 重构登录模块 #编程
```

勾选任务（将 `[ ]` 改为 `[x]`）后，系统会自动检测并采集药材。

### 兼容 Obsidian Tasks 插件

本插件完整兼容 [Obsidian Tasks](https://github.com/obsidian-tasks-group/obsidian-tasks) 的格式：

```markdown
- [x] 提交项目报告 #学术写作 📅 2026-04-10 ✅ 2026-04-08
- [x] 紧急修复 Bug #编程 🔺 📅 2026-04-08
- [ ] 准备周会 PPT #社交 ⏳ 2026-04-09
```

支持识别的格式标记：

| 标记 | 含义 | 对插件的影响 |
|:---:|------|------------|
| ✅ 日期 | 完成日期 | 记录完成时间 |
| 📅 日期 | 截止日期 | 判定守时引/延时引 |
| ⏳ 日期 | 计划日期 | 识别但不直接影响炼丹 |
| 🛫 日期 | 开始日期 | 识别但不直接影响炼丹 |
| ➕ 日期 | 创建日期 | 识别但不直接影响炼丹 |
| 🔺 / ⏫ | 高优先级 | 药材稀有度提升为珍材，数量翻倍 |
| 🔼 | 中优先级 | 识别但不额外加成 |
| 🔽 / ⏬ | 低优先级 | 识别但不额外加成 |

### 标签匹配规则

- 任务中的 `#标签名` 需要与你在设置中配置的**领域标签名称完全匹配**
- 例如：领域标签名为"学术写作"，则任务中需要写 `#学术写作`
- 如果任务没有匹配到任何领域标签，则不会采集药材
- 行内代码块（`` ` `` 包裹的内容）中的标签会被忽略

## 番茄钟

插件内置了番茄钟功能，通过侧边栏面板操作。

### 使用方式

1. 点击侧边栏 🔥 图标打开**丹道修炼面板**
2. 在面板中找到番茄钟区域
3. 操作按钮：

| 操作 | 说明 |
|------|------|
| **开始** | 启动一个新的番茄钟工作时段（默认 25 分钟） |
| **暂停** | 暂时停止计时，可随时继续 |
| **继续** | 从暂停处恢复计时 |
| **停止** | 提前结束当前番茄钟（视为中断） |

### 番茄钟流程

```
开始 → 工作时段（25分钟）→ 自动完成 → 获得专注引 🧘
                                    ↓
                             自动进入休息（5分钟）→ 休息结束 → 回到空闲
                             
手动停止（工作中）→ 获得断念引 💔
```

### 命令面板

你也可以通过命令面板（`Ctrl/Cmd + P`）快速操作：

- **开始番茄钟** — 立即启动番茄钟
- **停止番茄钟** — 停止当前番茄钟

## 每日封炉

### 自动封炉

- 默认每日 **23:00** 自动封炉
- 插件每分钟检查一次时间，到达封炉时间后自动执行
- 封炉后会弹出通知，告知炼成的丹药名称

### 补封机制

如果你某天没有打开 Obsidian（比如周末休息），插件在下次启动时会**自动补封**所有遗漏的日期，并弹出通知提示补封了多少天。

### 手动封炉

如果需要提前封炉（例如今天要早睡），有两种方式：

1. **设置页面**：进入 **设置 → 丹道修炼系统 → 高级设置**，点击 **立即封炉**
2. **命令面板**：按 `Ctrl/Cmd + P`，搜索 **手动封炉** 并执行

`dandao/修炼档案.md` 存储你的修炼境界、丹炉等级、经脉状态等核心数据。

`dandao/投注/` 文件夹存储项目、模板、角色以及投注记录数据。

### 种子与目标

- `dandao/种子池.md` 存储所有灵感种子
- `dandao/目标.md` 存储目标管理数据

### 周期炉数据

`dandao/周期炉/` 文件夹存储各长周期丹炉的数据。

### 每日修炼日志

`dandao/日志/YYYY-MM-DD.md` 记录每天的药材采集、药引和丹药信息，同样使用 frontmatter 存储结构化数据。

### data.json

插件的 `data.json`（位于 `.obsidian/plugins/dandao-cultivation/data.json`）只存储最小化的设置项和运行时状态，核心数据都在 Vault 的 Markdown 文件中。

## 设置项说明

进入 **设置 → 丹道修炼系统** 可配置以下选项：

### 基础设置

| 设置项 | 说明 | 默认值 |
|--------|------|:------:|
| 封炉时间 | 每日自动封炉的时间点（24 小时制） | `23:00` |
| 番茄钟工作时长 | 单个番茄钟工作时段的分钟数 | 25 分钟 |
| 番茄钟休息时长 | 番茄钟休息时段的分钟数 | 5 分钟 |

### 领域标签管理

| 操作 | 说明 |
|------|------|
| 标签列表预览 | 查看当前已配置的所有领域标签及其性味 |
| 打开配置文件 | 在 Obsidian 中直接编辑 `dandao/领域配置.md` |
| 新建标签 | 通过对话框快速创建新的领域标签 |

### 数据目录

| 设置项 | 说明 | 默认值 |
|--------|------|:------:|
| 数据存储路径 | 修炼数据在 Vault 中的目录名称（修改后需重启插件） | `dandao` |

### 显示设置

| 设置项 | 说明 | 默认值 |
|--------|------|:------:|
| 开炉动画 | 封炉后首次查看丹药时是否播放动画效果 | 开启 |
| 开炉音效 | 封炉后首次查看丹药时是否播放音效 | 关闭 |

### 高级设置

| 操作 | 说明 |
|------|------|
| 立即封炉 | 手动触发今日封炉 |

### 游戏参数

| 设置项 | 说明 | 默认值 |
|--------|------|:------:|
| 种子过期提醒天数 | 超过此天数未处理的种子显示提醒 | 7 天 |
| 药材有效期天数 | 药材采集后的默认有效期 | 7 天 |
| 突破难度 | 境界突破的难度（简单/普通/困难） | 普通 |
| 默认统计周期 | 统计 Tab 默认显示的时间范围 | 近一周 |

## 常见问题

### 为什么完成任务后没有采集到药材？

最常见的原因是**标签未匹配**。请检查：

1. 任务中是否添加了 `#标签名`（如 `#编程`）
2. 该标签名是否与设置中的领域标签名称**完全一致**
3. 任务格式是否正确（需要是 `- [x] 任务内容 #标签名` 的格式）
4. 标签是否写在了行内代码块中（`` `#标签` `` 会被忽略）

### 数据存储在哪里？

- 核心数据（领域配置、每日日志、丹药记录）存储在 Vault 根目录下的 `dandao/` 文件夹中，以 Markdown 文件形式保存
- 插件设置和运行时状态存储在 `.obsidian/plugins/dandao-cultivation/data.json` 中

### 如何备份数据？

由于所有核心数据都是 Vault 内的 Markdown 文件，只要你正常备份 Vault（例如使用 Git、iCloud、OneDrive 等），修炼数据就会一并备份。特别注意包含以下内容：

- `dandao/` 文件夹（全部修炼数据，含日志、投注、种子、周期炉等）
- `.obsidian/plugins/dandao-cultivation/data.json`（插件设置）

### 修改了封炉时间后为什么没有生效？

修改封炉时间后会立即生效，插件每分钟检查一次。如果当日已经封炉过，修改时间不会触发重复封炉。

### 插件支持移动端吗？

支持。本插件兼容 iOS 和 Android 上的 Obsidian 移动端。

### 侧边栏有哪些 Tab？

修炼面板共有 **5 个 Tab**：
- **药材**：查看今日采集的药材与药引
- **炼丹**：查看丹炉状态、丹药产出、丹炉等级
- **修炼**：查看境界进度、经脉状态、尝试突破
- **投注**：管理项目/模板/角色投注
- **种子**：管理灵感种子与目标
- **统计**：数据可视化与统计图表

### 药材过期了怎么办？

过期药材不再参与炼丹。建议及时完成任务消耗药材，或在设置中调整"药材有效期天数"延长保鲜时间。可以关注药材的陈化阶段颜色标识，合理安排炼丹节奏。

### 什么是"冲和"？怎么触发？

当丹炉中相冲性味（神识/根骨、灵动/融合）的药材比例恰好在 **40%~60%** 之间时，会触发"冲和"效果，丹药品级反而提升。这需要精心搭配当日的任务领域分布。

### 突破失败了怎么办？

不必担心，每次突破失败都会积累**心境值**，下次突破成功率更高。系统保证最终必定成功，坚持修炼即可。

### 投注的丹药还能回收吗？

丹药一旦投注即消耗，不可回收。请审慎选择投注方向，根据当前需求决定是投项目（短期加成）、模板（中期复利）还是角色（长期成长）。

### 多周期丹炉会消耗额外药材吗？

长周期丹炉（周炉、月炉、项目炉）会自动分配每日 **20%** 的药材，这部分药材不会重复参与日炉炼丹。
